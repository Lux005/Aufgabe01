Tipps und Tricks

Git Verbindung

  1. PS oder CMD öffnen
  2. Auf Webseite in das Projekt gehen (Repository)
  3. Dann der Clone Button drücken & Clone with SSH kopieren
  4. Beispiel; cd ./Downloads/ eingeben
  5. git clone + kopiertes eingeben und ausführen
  6. Beispiel; cd ./Projektname eingeben
  7. git add * eingeben (Entweder: * = Alle Datein, Oder: Dateiname)
  8. git status (Grün = Es wurde geaddet, Rot = Es wurde nicht geaddet)
  9. git commit -m &ldquo Kommentar &ldquo
  10. git push

Tipp: Mit cd.. kommt man aus dem aktuellen Ordner raus; Z.B. von /Downloads/Test/ zu /Downloads/


Branch

  • git branch --> Zeigt mit * an auf welchem Branch man sich befindet
  • git branch + Name --> Erstellt ein neues Branch
  • git checkout + Name --> Wechselt aktuellen Branch zu dem eingegebenen Branch
  • git push --set-upstream origin + Name --> Pusht neues Branch
  • git merge + Name --> Führt (mergt) eingegebenes Branch zum aktuellen Branch zusammen, also man holt sich die Datein aus dem angegebenem Branch
  • git branch -d + Name --> Löscht den angegebenen Branch
